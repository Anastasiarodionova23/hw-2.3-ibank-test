# Домашнее задание к занятию «Patterns» (Задача №2)

## Тестирование API интернет-банка

### Описание проекта
Автоматизированное тестирование API интернет-банка с использованием REST-assured. Проверяются сценарии создания пользователей через тестовый режим приложения.

### Технологии
- Java 11
- JUnit 5
- REST-assured
- Gson
- Gradle

### Запуск приложения
Перед запуском тестов необходимо запустить SUT в тестовом режиме:

```bash
  java -jar artifacts/app-ibank.jar -P:profile=test
```

### Приложение будет доступно по адресу:
http://localhost:9999

### Оценка времени тестирования:
### Ручное тестирование (80 минут) 
- проверка документации API
- Cоздание запросов в Postman
- Проверка различных комбинаций данных
- Анализ результатов
### Автоматизация (240 минут)
- Настройка проекта и зависимостей 
- Написание тестов
- Отладка и анализ ошибок
- Исследование реального поведения сервера
- Финальная стабилизация тестов

## Вывод 
Автоматизация заняла больше времени, но теперь у нас есть надежный набор тестов, который можно запускать одной командой. Это позволяет быстро проверять регрессию при изменениях в приложении и экономит время.
